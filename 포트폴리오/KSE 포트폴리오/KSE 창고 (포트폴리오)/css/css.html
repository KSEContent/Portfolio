1. 거의 모든 레이아웃 정렬 방식에 대해 알고 있습니다.

- 저는 HTML & CSS를 위주로 공부를 했습니다. 처음에는 나열 방식에 대해 공부를 했습니다. 그렇게 현재는 float, flex, column-count, grid 형식의 레이아웃을 정렬 할 줄 압니다. 지금은 많은 사이트를 돌아본 결과 float가 많이 쓰이는 것을 발견했고 Caniuse 검색을 통해 브라우저 구버전과 현재까지 폭넓은 지원을 받는다는걸 확인해 float를 위주로 사용하고 있습니다.

2. 가상 선택자에 능숙합니다.

- 가상 선택자들은 저의 친구들 입니다. 저는 자바스크립트의 도움을 최대한 받지 않고 CSS를 위주로 만든 페이지들이 많습니다. 특히 ::before 와 ::after 의 사용에 능숙하며 다른 선택자도 능숙하게 활용할 줄 압니다.

3. 중복 사용이 가능한 CSS파일을 따로 저장해 놓습니다.

- 예전에는 헤드와 푸터부분도 복붙으로 활용하는 경우가 많았는데, 현재는 따로 중복 사용이 가능한 CSS파일을 만들어 외부링크로 연결해 놓습니다. 위에 이미지는 미디어 쿼리며 하나의 미디어 쿼리를 만들어 전체 틀을 반응형으로 만들었습니다.
